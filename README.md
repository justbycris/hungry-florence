# Hungry Florence

*Hungry Florence* it's a 2D game developed with HTML, CSS, Phaser.JS and Claude. The main character is *surprised* Florence! A beautiful cow cat living in the streets of New York City (based on a true story). 

Florence loves living in NYC, and loves watching birds, squirrels and fish videos. She also loves to miau in a strange way (play the game to find out!).  

Game instructions: 
Use your left and right arrow keys to move left and right or 'A' and 'B'.
